# Manga Reader 
<img width="100" height="100" alt="cReddit" src="https://github.com/pawanabc59/mangaReader/blob/master/mangaReader/app/src/main/res/drawable-v24/luffy_icon.png">

  An application for Android users to read E-Comics and E-Books at one place. It is an app designed basically for the Manga (comic) lovers so they can read there favourite comics without any bother at one place.

# Features:
    • Dark Mode in the App.
    • Different sections for manga and books.
    • Users can read the manga/book and can also add that to their favourites.
    • Users can rate each chapter of manga that they read.
    • Modern UI for my profile.
    • Users can track their recently read manga.
